# Model Truck Platooning

### Requirements
- ROS Kinetic

### Setup

#### 1. Create workspace
	$ mkdir -p ~/catkin_ws/src
	$ cd ~/catkin_ws/
	$ catkin_make
#### 2. Clone repository
	$ cd ~/catkin_ws/src
	$ git clone https://github.com/quilder/modeltruck-platooning.git modeltruck_platooning
	$ cd ..
	$ catkin_make
	$ source devel/setup.bash
